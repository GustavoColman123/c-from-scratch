## Digit Histogram (Extension Beyond K&R)

### Context

This histogram is not explicitly implemented in *The C Programming Language* by Kernighan and Ritchie. Instead, it is an educational extension derived from the digit frequency example in Chapter 1.

The goal is to deepen the understanding of arrays and data representation by transforming numerical frequencies into a visual format.

### Purpose

- Reinforce the use of arrays as frequency tables.
- Introduce the concept of data visualization in a terminal environment.
- Practice proportional scaling and formatted output.
- Demonstrate the transition from data accumulation to data representation.
